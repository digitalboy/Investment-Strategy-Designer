

# 项目名称：VestLab ETF 策略设计器 (ETF Strategy Designer)

## 1. Slogan (口号)

**主标题：拒绝凭感觉，用数据说话。**
*(Stop Guessing. Start Validating.)*

**副标题：将您的“抄底直觉”转化为可验证的投资策略。无需写代码，几秒钟内完成十年历史回测。**

---

## 2. 核心价值观 (Core Values)

我们不仅提供工具，更倡导一种理性的投资哲学。

### 🎯 信号驱动，而非日历驱动 (Signal over Schedule)
传统的定投是“到时间就买”，无论贵贱。我们主张**“智能定投”**——当市场发出“打折”信号（如大跌、RSI超卖）时果断出手，在市场狂热时保持冷静。让每一分钱都花在刀刃上。

### 🧠 化繁为简，人人可用 (Simplicity in Complexity)
量化交易不应是程序员的专利。我们将复杂的算法封装在**“自然语言”**的交互中。您不需要懂 Python，只需要像说话一样告诉系统：“如果跌了15%，就买入1000元”，剩下的交给我们。

### ⚖️ 诚实对比，理性归因 (Honest Benchmarking)
策略好不好，比过才知道。我们将您的策略与**“买入并持有”**及**“无脑定投”**进行同台竞技。不仅看收益率，更看**最大回撤**。我们帮助您区分运气与实力，找到风险收益比最优的平衡点。

---

## 3. 核心功能亮点 (Key Features)

### ✨ 意图导向的策略构建 (Intent-Based Builder)
告别晦涩的参数面板。
*   **直观场景**：直接选择“大跌抄底”、“追涨突破”或“趋势跟随”。
*   **自然语言填空**：“当价格从高点下跌 `15%` 时，买入 `20%` 仓位”。所见即所得，配置策略就像填空题一样简单。

### 🚀 极速回测引擎 (Lightning-Fast Backtest)
基于 Cloudflare 全球边缘网络。
*   **秒级响应**：无需漫长等待，数千个交易日的数据运算在毫秒级完成。
*   **全球数据**：覆盖 SPY, QQQ, NVDA 等全球主流 ETF，数据源于雅虎财经，每日更新。
*   **精准模拟**：严格遵循“次日开盘执行”原则，杜绝“未来函数”作弊，还原真实的交易环境。

### 📊 深度可视化报告 (Visual Performance Analytics)
一图胜千言。
*   **净值曲线对比**：您的策略 vs 基准指数，超额收益一目了然。
*   **买卖点标记**：在 K 线图上清晰标记每一次自动买入和卖出的位置，复盘您的每一次决策。
*   **风险透视**：不仅展示年化回报，更突出**最大回撤 (Max Drawdown)**，让您清楚知道极端行情下账户可能缩水多少。

### 🛡️ 智能风控 (Smart Risk Control)
*   **冷却期机制**：防止在下跌途中过早打光子弹，控制交易频率。
*   **资金管理**：支持按“固定金额”、“现金比例”或“总资产比例”动态调整仓位，实现真正的金字塔式建仓。

---

## 4. 用户旅程 (User Journey - for Landing Page)

1.  **定义想法**：您觉得 QQQ 每次跌 10% 都是机会？
2.  **构建策略**：在设计器中输入规则：`IF` 从高点回撤 10% `THEN` 买入 $1000。
3.  **验证历史**：一键运行。系统告诉您：如果过去 10 年这么做，您的收益是 300%，比死拿多赚了 50%，且回撤更小。
4.  **增强信心**：看着数据，您不再恐慌下跌，而是期待机会。
